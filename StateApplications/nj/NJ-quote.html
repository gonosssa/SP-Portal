New Jersey Temporary Disability Benefits Application

Complete the DP-1 form to apply for approval or modification of your insured Private Plan.

This application is for the New Jersey Temporary Disability Benefits (TDI) program. The DP-1 form is used to:

  • Request approval for a new Private Plan
  • Request modification to an existing Private Plan
  • Transfer coverage from the State Insurance Fund to a Private Plan

Note: This application will be submitted to the Division of Temporary Disability Insurance, PO Box 957, Trenton, New Jersey 08625-0957.

What You'll Need

  • New Jersey Employer Identification Number (9-digit number assigned by Division of UI/DI Financing)
  • Private Plan Number (if modifying an existing plan)
  • Employer information (name, address, phone)
  • Authorized representative information
  • Number of New Jersey employees
  • Plan coverage and benefit details
